# Project-Magang-Peta-Informasi-UPTD-Wilayah-Sungai-Cisadea-Cibareno
Peta informasi Unit Pelaksana Teknis Daerah (UPTD) wilayah sungai Cisadea–Cibareno.

## Deskripsi
Repository ini hasil kegiatan magang yang dilakukan di Dinas Sumber Daya Air Provinsi Jawa Barat. Peta informasi UPTD Wilayah Sungai Cisadea–Cibareno menampilkan cakupan
wilayah kerja UPTD serta informasi umum wilayah sungai. Informasi yang disajikan meliputi batas administrasi, batas wilayah sungai, batas daerah aliran sungai, dan elemen pendukung lainnya sebagai bahan visualisasi dan penyajian data wilayah sungai.

## Kegiatan
- Pengolahan dan penyajian data spasial
- Penyusunan peta informasi wilayah

## Tools
QGIS

## Output
Peta informasi UPTD Wilayah Sungai Cisadea–Cibareno
